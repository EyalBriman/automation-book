## הוספת שאלה חדשה

כל שאלה כתובה בדפוס הבא בקובץ ה-`.qmd` המתאים:

```markdown
::: {.exercise}
טקסט השאלה. ניתן לכלול LaTeX: $x^2 + y^2 = 1$
ותמונות: ![תיאור](../images/my_image.png)
:::

::: {.solution}
הפתרון המפורט מופיע כאן.
הכפתור "הצג פתרון" נוצר אוטומטית.
:::
```

## עדכון הספר

```bash
git add .
git commit -m "add questions"
git push
```

הספר מתעדכן אוטומטית ב-GitHub Pages תוך ~2 דקות.
