## ➕ הוספת שאלה חדשה

העתיקו את הבלוק הבא לתוך הקובץ המתאים, **בכל מקום שבו תרצו** (אין סדר חשוב):

```markdown
::: {.exercise}
כתבו כאן את השאלה. ניתן לכתוב עברית חופשית, LaTeX, ולהוסיף תמונות.
:::

::: {.solution}
כתבו כאן את הפתרון המפורט.
הכפתור "הצג פתרון" נוצר אוטומטית — אין צורך לעשות כלום נוסף.
:::
```

> **שימו לב:** חייבים להשאיר שורה ריקה **לפני ואחרי** כל `:::`.

## 📐 הוספת נוסחאות LaTeX

**בתוך שורה:**
```markdown
נוסחה קצרה: $G(s) = \frac{1}{s+1}$
```

**בשורה נפרדת (מרכזית):**
```markdown
$$
\omega_n = \sqrt{\frac{k}{m}}, \qquad \zeta = \frac{c}{2\sqrt{km}}
$$
```

**דוגמאות שימושיות:**

| מה רוצים | קוד LaTeX |
|----------|-----------|
| שבר | `\frac{a}{b}` |
| כוח | `x^{2}` |
| שורש | `\sqrt{x}` |
| אינטגרל | `\int_0^T f(t)\,dt` |
| גבול | `\lim_{s \to 0}` |
| מטריצה | `\begin{pmatrix} a & b \\ c & d \end{pmatrix}` |
| יחידות | `10\,\text{ms}` |

## 🖼️ הוספת תמונה

1. **שמרו את קובץ התמונה** בתיקייה `images/` (צרו אותה אם לא קיימת).
2. **הפנו אליה** בתוך השאלה או הפתרון:

```markdown
![כיתוב התמונה](../images/שם_הקובץ.png){width=70%}
```

- `width=70%` — שנו לפי הצורך (50%, 80%, 100%)
- פורמטים נתמכים: `.png`, `.jpg`, `.jpeg`, `.svg`
- הכיתוב מוצג מתחת לתמונה

**דוגמה מלאה עם תמונה:**
```markdown
::: {.exercise}
בהתאם לסכמה הבאה:

![סכמת המעגל](../images/circuit_diagram.png){width=60%}

חשבו את קבוע הזמן $\tau$.
:::

::: {.solution}
מהסכמה ניתן לראות כי $R = 1\,k\Omega$ ו-$C = 10\,\mu F$, לכן:

$$\tau = RC = 10\,\text{ms}$$
:::
```

לאחר כל עריכה, הריצו בטרמינל:
